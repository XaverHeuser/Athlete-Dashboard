# Strava Integration Documentation

## Overview

This section provides references and resources for integrating with the Strava API, which allows access to user activity data, profiles, segments, and more.

## API Reference

The complete official Strava API documentation can be found here: [Strava Developer Reference](https://developers.strava.com/docs/reference/)

## Notes

- You will need a Strava Developer Account to obtain client credentials.
- API access requires OAuth 2.0 authentication.
- Rate limits apply; refer to the Strava documentation for current limits.
